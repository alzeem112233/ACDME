# Al-Qaisar Master Dashboard

تطبيق إدارة أكاديمية رياضية مبني بواجهة عربية RTL وقابل للنشر كتطبيق ويب.

## التشغيل

```bash
pnpm install
pnpm dev
```

## البناء

```bash
pnpm build
```

## الوحدات

- تسجيل الدخول و OTP
- إعدادات الأكاديمية
- إدارة الفئات العمرية
- إدارة الفرق
- إدارة اللاعبين
- ملف اللاعب
- ملف الفريق
- متابعة الحضور والمدفوعات
- الإدارة المالية
- التقارير
- التحفيز والمكافآت
- تسجيل نتائج المباراة
- التنبيهات
- مركز المجتمع

## النشر

المشروع مجهز للنشر على Vercel أو Netlify أو Docker.
راجع [docs/DEPLOYMENT.md](docs/DEPLOYMENT.md).
