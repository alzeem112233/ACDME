# التحويل إلى أندرويد

المشروع مجهز حالياً كتطبيق ويب قابل للتثبيت PWA. هذا يعني أنه يمكن استخدامه من الهاتف كتطبيق مستقل من المتصفح.

عند اعتماد النسخة الأولى، يوجد مساران:

## المسار السريع

تغليف التطبيق باستخدام WebView أو Capacitor.

مناسب عندما تكون أغلب الوظائف داخل الويب:

- إدارة اللاعبين
- الحضور
- المدفوعات
- التقارير
- التنبيهات داخل التطبيق

## المسار الأقوى

تطبيق أندرويد أصلي أو React Native لاحقاً.

مناسب إذا احتجنا:

- إشعارات نظام متقدمة
- عمل offline واسع
- كاميرا ورفع مستندات
- صلاحيات هاتف حساسة

## ما يلزم قبل التحويل

1. تثبيت قاعدة البيانات والسيرفر.
2. تسجيل دخول حقيقي برقم الهاتف و OTP.
3. تحديد رابط الإنتاج النهائي.
4. اختبار التطبيق على قياسات هواتف مختلفة.
